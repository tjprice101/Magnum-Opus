================================================================================
  RESURRECTION OF THE MOON -- GRADIENT ASSETS
  Midjourney Prompt Reference
================================================================================

PURPOSE:
  Horizontal gradient strip textures sampled by pixel/vertex shaders to
  color projectile trails, beam fills, explosion shockwaves, and particle
  lifetime color ramps. The shader reads a U coordinate (0.0 to 1.0) along
  the strip width to determine color at each pixel of the rendered effect.

USAGE IN MOD:
  - Trail shader gradient map: ShaderLoader samples these to color the
    comet trail from core to wake (left=tip/bright, right=tail/cold)
  - Explosion shockwave ring color: shader reads radially from center to edge
  - Particle lifetime color ramp: newborn particles sample left, dying
    particles sample right
  - Beam fill gradient: beam center samples left edge, beam border samples
    right edge
  - Resurrection ultimate ability aura color sweep

FORMAT REQUIREMENTS:
  - 256x16 or 512x32 PNG (wide horizontal strip)
  - Seamless is NOT required (these are clamped, not tiled)
  - Left edge = brightest / most intense / core color
  - Right edge = coolest / darkest / ambient color
  - Vertical variation is minimal (subtle noise/texture OK but keep it
    mostly uniform vertically so shader V-coordinate does not matter much)
  - sRGB color space, 8-bit per channel

COLOR PALETTE:
  Deep Space Violet ........... RGB(50, 20, 100)
  Impact Crater Indigo ........ RGB(100, 80, 200)
  Comet Trail Lavender ........ RGB(180, 120, 255)
  Lunar Shine Blue ............ RGB(120, 190, 255)
  Comet Core Pale Blue ........ RGB(210, 225, 255)
  Blue-White Impact ........... RGB(235, 240, 255)

================================================================================
  POST-GENERATION NOTES:
  - After generation, resize to exactly 512x32 or 256x16 in Photoshop/GIMP
  - Use bicubic interpolation to prevent banding during resize
  - Verify no pure black pixels exist in the left 25% of any gradient (shader
    would read these as invisible in additive mode)
  - Test each gradient in ShaderLoader by assigning it as the gradient map and
    verifying the trail/beam/particle coloring reads correctly
  - The Resurrection Aura Sweep (#4) is symmetric; if needed, the shader can
    ping-pong sample it for a pulsing effect
  - For Beam Fill (#5): the rightmost pixels MUST be RGB(0,0,0) or the beam
    will have a visible hard edge in-game
================================================================================
