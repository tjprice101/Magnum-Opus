================================================================================
  RESURRECTION OF THE MOON -- TRAILS/CLEAR ASSETS
  Midjourney Prompt Reference
================================================================================

PURPOSE:
  Elongated horizontal trail strip textures identical in concept to the
  Trails/ folder but rendered with fully TRANSPARENT backgrounds and soft
  alpha-feathered edges instead of black backgrounds. These are used with
  alpha-blended (non-additive) rendering passes where a black background
  would show as visible darkening rather than being invisible.

USAGE IN MOD:
  - Overlay trail rendering pass (drawn AFTER the scene, not additively)
  - Semi-transparent comet wake that partially obscures what is behind it
    rather than brightening it (gives a smoky/dusty quality)
  - Impact distortion trail where the effect needs to darken or tint the
    background rather than brighten it
  - Layered trail compositing: the Clear trail is drawn first as a base,
    then the additive (Trails/) version is drawn on top for bloom
  - Resurrection aura sweep that wraps around the player with proper
    alpha-based opacity rather than additive glow

FORMAT REQUIREMENTS:
  - 256x64 or 512x128 PNG with ALPHA CHANNEL (32-bit RGBA)
  - Background must be fully transparent (alpha = 0) NOT black
  - The trail visuals exist purely in the RGB+A channels
  - Edges must feather to alpha=0 smoothly (no hard cutoffs)
  - Top edge, bottom edge, and right edge all fade to full transparency
  - Premultiplied alpha (PMA) preferred for tModLoader compatibility
  - sRGB color space, 8-bit per channel

COLOR PALETTE:
  Deep Space Violet ........... RGB(50, 20, 100)
  Impact Crater Indigo ........ RGB(100, 80, 200)
  Comet Trail Lavender ........ RGB(180, 120, 255)
  Lunar Shine Blue ............ RGB(120, 190, 255)
  Comet Core Pale Blue ........ RGB(210, 225, 255)
  Blue-White Impact ........... RGB(235, 240, 255)

  NOTE: Because these use alpha transparency instead of additive blending,
  the colors can include darker and more saturated tones that would be
  invisible in additive mode. The deep violets and indigos will actually
  be VISIBLE here, which gives these trails a richer, moodier appearance.

================================================================================
  POST-GENERATION NOTES:
  - CRITICAL: Midjourney DOES NOT generate true transparent PNGs. You MUST
    manually process every generated image:
    1. Generate the image from Midjourney (it will have a visible background)
    2. In Photoshop/GIMP, carefully remove the background to create alpha
    3. Use "Select > Color Range" on the background color, then delete
    4. For smoky/cloudy trails, this may require using "Blend If" sliders
       or manual masking rather than simple color keying
    5. For the ember particles (#4), each spark should retain its glow halo
       in the alpha channel
  - Alternative workflow: Use the Midjourney output as a REFERENCE, then
    paint the trail manually in Photoshop with a transparent canvas, using
    the generated image as a visual guide for color placement and flow
  - Export as 32-bit RGBA PNG (512x128 preferred, 256x64 minimum)
  - Convert to premultiplied alpha for tModLoader compatibility:
    R_out = R * A, G_out = G * A, B_out = B * A, A_out = A
  - Verify in-engine: draw the Clear trail over a bright white background
    AND a dark background to confirm it reads correctly in both contexts
  - Layer compositing order in the draw call should be:
    1. Draw Clear trail (alpha blend) as base layer
    2. Draw additive Trail (from Trails/ folder) on top for bloom/glow
  - The Clear versions should be slightly WIDER vertically than their
    additive counterparts so the smoky/dusty base extends beyond the
    bright glow center
