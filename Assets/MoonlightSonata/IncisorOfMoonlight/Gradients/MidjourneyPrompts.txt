================================================================================
 INCISOR OF MOONLIGHT — GRADIENT TEXTURES
 Midjourney Prompt Reference
================================================================================

PURPOSE:
  Horizontal color gradient strips sampled by HLSL pixel shaders during trail
  and beam rendering. The IncisorResonance.fx shader uses these as lookup
  textures (sampled via coords.x along the trail) to determine trail color at
  each pixel. The MoonlightTrail.fx shader also samples gradients for the
  multi-pass glow rendering system.

  The CalamityStyleTrailRenderer renders 3 passes per trail, each sampling
  different regions of the gradient. The shader's gradientT variable maps
  trail position (head=0, tail=1) into the gradient texture's U coordinate.

  These are NOT decorative images — they are functional data textures that
  shaders read pixel-by-pixel to determine color output.

TECHNICAL REQUIREMENTS:
  - Dimensions: 256x16 (primary) or 512x32 (high-detail variant)
  - Horizontal orientation: Left edge = trail head (newest), Right edge = tail
  - Vertically uniform: Each column should be the same color top-to-bottom
    (or with very subtle vertical variation for noise)
  - Format: PNG, no alpha needed (fully opaque)
  - Clean smooth blending: No hard edges, banding, or dithering
  - NO text, NO symbols, NO patterns — pure smooth color gradients only
  - Must display the EXACT colors specified — shader reads literal RGB values

 POST-PROCESSING NOTES:
  - CRITICAL: Resize outputs to exact dimensions (256x16 or 512x32)
  - Verify vertical uniformity — each pixel column must be one color
  - Remove any noise, grain, or texture artifacts from generation
  - Check that the leftmost and rightmost pixel columns match the specified
    endpoint colors exactly — the shader reads these literal values
  - Save as PNG without alpha (or with fully opaque alpha)
  - Test by zooming in to 1600%+ to verify there is no visible banding or
    stepping in the gradient transitions
  - These may need manual correction in an image editor to achieve the
    mathematically smooth gradients that shader sampling requires
  - The IncisorResonance.fx shader samples these with:
      gradientT = coords.x * 0.7 + standingWaveVal * 0.3
    so the middle 70% of the gradient gets the most use
