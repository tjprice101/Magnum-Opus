================================================================================
  ETERNAL MOON — GRADIENT TEXTURES
  Midjourney Prompt Reference
================================================================================

PURPOSE:
  Horizontal gradient strip textures sampled by pixel/vertex shaders to color
  trails, swing arcs, and tidal wave projectile effects. The shader reads a UV
  coordinate (usually the progress along the trail or swing arc) and samples
  color from left-to-right across the gradient strip.

USAGE IN CODE:
  - CalamityStyleTrailRenderer.TrailStyle.Cosmic trail shader (samples gradient
    for trail color based on distance from blade tip)
  - EternalMoonVFX.GetLunarPhaseColor() visual reference — the gradient encodes
    the DeepTide -> DarkPurple -> Violet -> IceBlue -> MoonWhite color ramp
  - EternalMoonVFX.WaveTrailColor() for projectile trail rendering
  - ScrollingTrailShader.fxc and CelestialValorTrail.fxc gradient sampling
  - Tidal wash projectile (EternalMoonTidalWash) arc coloring
  - BeamGradientFlow.fxc for beam projectile color ramping

DIMENSIONS: 256x16 or 512x32 horizontal strips (width is sample resolution,
  height provides anti-aliasing margin and vertical gradient variation)
FORMAT: Opaque (no transparency needed — shaders sample RGB only)

COLOR PALETTE (left-to-right flow represents shadow-to-light progression):
  - Night Void:     RGB(40, 10, 60)    — far left, trail end / shadow
  - Deep Indigo:    RGB(75, 0, 130)    — early transition
  - Violet Body:    RGB(138, 43, 226)  — mid-body primary
  - Ice Blue Flare: RGB(135, 206, 250) — bright zone
  - Glacial Blue:  RGB(170, 225, 255) — icy accent peak
  - Moon White: RGB(230, 235, 255) — far right, cool core

REFERENCE: SandboxLastPrism/Gradients/RainbowGrad1.png
