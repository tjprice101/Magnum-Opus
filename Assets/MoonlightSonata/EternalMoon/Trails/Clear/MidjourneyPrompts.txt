================================================================================
  ETERNAL MOON — CLEAR TRAIL TEXTURES (Transparent Background)
  Midjourney Prompt Reference
================================================================================

PURPOSE:
  Same conceptual content as the Trails/ textures but rendered with fully
  transparent backgrounds and soft feathered alpha edges. These textures are
  used for OVERLAY trail rendering where the trail is composited on top of
  the game scene using alpha blending rather than additive blending. The alpha
  channel controls visibility, whereas regular Trails/ textures rely on
  black = invisible via additive blend math.

USAGE IN CODE:
  - Used when CalamityStyleTrailRenderer switches to non-additive blend states
  - GlowTrailClear-style rendering for soft overlay sword arcs
  - ThinLineGlowClear-style rendering for thin connecting flow lines between
    wake points in DrawTidalWake()
  - Overlay compositing for motion blur passes (MotionBlurBloom.fxc)
  - Cases where the trail needs to partially occlude what is behind it rather
    than purely brighten it (semi-opaque tidal flow effects)

DIMENSIONS: 256x64 or 512x128 horizontal strips. Same left-bright to
  right-fading directionality as regular Trails.

CRITICAL DIFFERENCE FROM TRAILS/:
  - Background MUST be fully transparent (alpha = 0), not black
  - Trail energy itself has partial alpha (soft feathered edges blend to zero)
  - Colors can include darker tones that would be invisible in additive mode
  - The texture edges (top, bottom, right) must fade smoothly to zero alpha
  - Left edge can be fully opaque
  - Midjourney does not natively support transparency, so generate on WHITE
    background and note that post-processing will convert white to transparent.

POST-PROCESSING NOTE:
  Midjourney outputs will need post-processing in Photoshop/GIMP:
  1. Use "Color Range" to select the white background
  2. Delete selected pixels to create transparency
  3. Apply slight Gaussian blur to alpha channel edges for smooth feathering
  4. Save as PNG with alpha channel

COLOR PALETTE:
  - Night Void:     RGB(40, 10, 60)    — semi-opaque trailing dark tones
  - Deep Indigo:    RGB(75, 0, 130)    — body shadow tones
  - Violet Body:    RGB(138, 43, 226)  — primary trail color
  - Ice Blue Flare: RGB(135, 206, 250) — bright highlights
  - Glacial Blue:  RGB(170, 225, 255) — icy accent streaks
  - Moon White: RGB(230, 235, 255) — leading edge core

REFERENCE: SandboxLastPrism/Trails/Clear/GlowTrailClear.png,
  ThinLineGlowClear.png
