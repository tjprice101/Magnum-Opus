# Noise Texture Prompts for Radial Scroll VFX

> **PURPOSE**: Midjourney prompts for generating seamless tileable noise textures optimized for radial scroll shaders and theme-specific VFX effects.
>
> **REQUIREMENTS**: All textures must be:
> - **Seamlessly tileable** (edges wrap perfectly)
> - **Grayscale or single-channel RGB** (shader tints colors)
> - **512x512 or 1024x1024 resolution**
> - **PNG format with no compression artifacts**

## üîß POST-PROCESSING TIPS

After generating textures in Midjourney:

1. **Check seamless tiling**: Open in image editor, offset 50% both axes, fix any visible seams
2. **Convert to grayscale**: Use desaturation or channel mixer for consistent values
3. **Adjust contrast**: Ensure full value range (0-255) for maximum shader control
4. **Resize if needed**: 512x512 is standard, 1024x1024 for high-detail effects
5. **Save as PNG**: No compression artifacts, preserve value precision
