================================================================================
NOISE TEXTURES & MESH ASSETS FOR SHADER-BASED VFX
================================================================================

PURPOSE: These textures are NOT meant to be seen directly. They are DATA textures
used by shaders to create smooth, organic visual effects without visible edges.

CRITICAL REQUIREMENTS FOR ALL NOISE TEXTURES:
- SEAMLESS/TILEABLE (must loop perfectly in all directions)
- GRAYSCALE only (black to white, no color)
- High contrast range (full 0-255 spectrum utilized)
- PNG format, NO compression artifacts
- Recommended sizes: 256x256, 512x512, or 1024x1024
- NO text, NO watermarks, NO borders

SHADER USAGE EXAMPLE:
```hlsl
// Sample noise for fog effect
float noiseValue = tex2D(noiseSampler, uv + time * scrollSpeed).r;
float alpha = smoothstep(dissolveThreshold, dissolveThreshold + 0.1, noiseValue);
```

POST-PROCESSING IN PHOTOSHOP/GIMP:
1. Convert to grayscale if not already
2. Adjust levels to use full 0-255 range
3. Use "Offset" filter to check seamless tiling
4. Clone stamp any visible seams
5. Save as PNG with no compression
