## ‚öôÔ∏è UNIVERSAL SPRITE RULES

### Minion Sprite Requirements
- **Size**: 32x32, 36x44, or 40x50 pixels (varies by minion complexity)
- **Style**: Terraria pixel art aesthetic, 16-bit retro game feel
- **Background**: Transparent PNG (generate on black, extract later)
- **Orientation**: Facing RIGHT (will be flipped in-game for left)
- **Color**: Full color, vibrant, themed to their musical score

### Buff Icon Requirements
- **Size**: 32x32 pixels exactly
- **Style**: Terraria buff icon style - clear, readable at small size
- **Background**: Transparent PNG
- **Border**: Subtle dark outline for readability
- **Symbol**: Should represent the minion or its effect clearly

## üé® POST-PROCESSING NOTES

After generating in Midjourney:

1. **Remove black background** ‚Üí Make transparent PNG
2. **Downscale** to target pixel size with nearest-neighbor interpolation (no blur)
3. **Check sprite bounds** - ensure centered and fitting target dimensions
4. **Test in-game** with color tinting if needed
5. **Place files** in correct Content/[Theme]/Projectiles/ or Content/[Theme]/Weapons/ folders

### File Placement:
- Minion sprites go **next to their .cs file**
- Buff icons go **next to their buff .cs file** (same folder)
- Example: `FlowerSpriteMinion.cs` ‚Üí `FlowerSpriteMinion.png` in same folder
