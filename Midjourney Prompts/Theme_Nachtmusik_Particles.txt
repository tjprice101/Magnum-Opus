# Nachtmusik - Particles and Effects

## Theme Overview

Nachtmusik (Eine kleine Nachtmusik - A Little Night Music) theme represents the serene night sky, starlight, celestial beauty, and nocturnal elegance. Visual motifs include:
- **Stars** - twinkling, various sizes, patterns
- **Constellations** - connected star patterns
- **Night sky elements** - meteors, comets, aurora
- **Moonlight** - gentle lunar glow
- **Stardust** - cosmic sparkle particles
- **Night creatures** - subtle hints of owls, moths

## Color Palette Reference
```
NachtDeepBlue    - (20, 30, 80)   - Deep night sky
NachtMidnight    - (40, 50, 120)  - Midnight blue
NachtStarlight   - (200, 210, 255) - Star white-blue
NachtGold        - (255, 220, 150) - Warm star gold
NachtSilver      - (220, 225, 240) - Cool moonlight
NachtPurple      - (80, 60, 140)  - Twilight purple
```

## Implementation Notes

When tinting these particles in code, use the Nachtmusik gradient:
```csharp
// Nachtmusik gradient: Deep Blue → Midnight → Starlight
Color GetNachtGradient(float progress)
{
    if (progress < 0.5f)
        return Color.Lerp(NachtDeepBlue, NachtMidnight, progress * 2f);
    else
        return Color.Lerp(NachtMidnight, NachtStarlight, (progress - 0.5f) * 2f);
}

// For warm star accents:
Color warmStar = Color.Lerp(NachtStarlight, NachtGold, 0.5f);

// For twilight effects:
Color twilight = Color.Lerp(NachtPurple, NachtMidnight, progress);
```

Combine multiple particle types for night sky effects:
- Stars + stardust for ambient sparkle
- Constellations + trails for projectiles
- Meteors + comets for powerful attacks
- Aurora + moonlight for area buffs
