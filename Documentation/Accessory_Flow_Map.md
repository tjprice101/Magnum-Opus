## 1) Core Chain Crafting Trees

### 1.1 Melee

```mermaid
flowchart TD
    RRB[ResonantRhythmBand] --> STC[SpringTempoCharm]
    STC --> SCR[SolarCrescendoRing]
    SCR --> HRS[HarvestRhythmSignet]
    HRS --> PCS[PermafrostCadenceSeal]
    PCS --> VTM[VivaldisTempoMaster]

    FCS[FatesCosmicSymphony] --> NSB[NocturnalSymphonyBand]
    NSB --> IFT8[InfernalFortissimoBandT8]
    IFT8 --> JCB[JubilantCrescendoBand]
    JCB --> ERB[EternalResonanceBand]

    NSB --> SJG[StarfallJudgmentGauntlet]
    IFT8 --> SJG
    SJG --> TCG[TriumphantCosmosGauntlet]
    JCB --> TCG
    TCG --> GES[GauntletOfTheEternalSymphony]
    ERB --> GES
```

Notes:
- T7-T10 are theme-energy gated (Nachtmusik, Dies Irae, Ode to Joy, Clair de Lune).
- T7 does not require VivaldisTempoMaster in recipe; it keys off FatesCosmicSymphony progression stage.
